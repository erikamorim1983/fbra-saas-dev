# N0T4X Brand & Design Tokens

Este documento registra a paleta de cores e diretrizes visuais oficiais da plataforma N0T4X para garantir consistência em futuras atualizações.

## Paleta de Cores (CSS Variables)

| Token | Hex/Valor | Uso Principal |
| :--- | :--- | :--- |
| **Primary (Brand Blue)** | `#081F2E` | Backgrounds escuros, sidebar, cabeçalhos, botões principais. |
| **Primary Light** | `#163a54` | Variações suaves e hover em elementos escuros. |
| **Primary Dark** | `#041018` | Fundo do Portal Adm (Dark Mode) e sombras profundas. |
| **Accent (Gold/Yellow)** | `#d4af37` | Itens ativos, CTAs secundários, sublinhados de destaque. |
| **Accent Light** | `#e6c86e` | Hover em elementos Accent. |
| **Neon (Highlight)** | `#00ffcc` | Detalhes tecnológicos, micro-interações, ícones sociais no hover. |
| **Background** | `#fdfdfd` | Fundo das páginas (Light Mode). |
| **Foreground** | `#081F2E` | Texto principal em fundos claros. |

## Estilos Complexos

- **Glassmorphism:** `bg-white/70 backdrop-blur-md border border-primary/5 shadow-xl`
- **Text Gradient:** From: `Primary` (#081F2E) to `Neon` (#00ffcc).
- **Sidebar (Admin/Dashboard):**
    - Background: `Primary` (#081F2E)
    - Active Item: `Background: Accent (#d4af37)`, `Text: Primary (#081F2E)`
    - Non-active Item: `Text: White/60`, `Hover BG: White/5`

## Tipografia
- **Principal:** 'Inter', sans-serif.
- **Peso:** Bold (700) e Black (900) para títulos.
